# Education Insights Dashboard: Enrollment, Literacy, and School Infrastructure

## Project Overview
This project presents an interactive data visualization dashboard that analyzes key education indicators across Indian States and Union Territories. The dashboard was created using Microsoft Power BI to transform raw educational data into meaningful visual insights.

The analysis focuses on indicators such as Gross Enrollment Ratio (GER), literacy rate, number of schools, and school management types over multiple financial years.

## Dataset
The dataset used in this project was obtained from the **National Data and Analytics Platform (NDAP)** provided by NITI Aayog.
Source: https://ndap.niti.gov.in

The dataset contains education-related indicators for **States and Union Territories of India** from **Financial Year 2018–19 to 2022–23**.

### Dataset Features
- Country
- State / Union Territory
- Financial Year
- Type of School Management
- Gender
- GER (Primary, Upper Primary, Secondary, Higher Secondary)
- Literacy Rate
- Number of Schools

## Tools and Technologies
- Microsoft Power BI
- Microsoft Excel
- Data Visualization Techniques

## Dashboard Features
The dashboard includes the following visualizations:

- Card visuals displaying key metrics
- Number of Schools by Financial Year
- Literacy Rate distribution by gender
- Gross Enrollment Ratio comparison across education levels
- Gross Enrollment Ratio comparison based on school management
- Interactive slicers for filtering by year, state/UT, gender, and school management

## Key Insights
- Primary education shows higher enrollment compared to secondary levels.
- Literacy rates vary across different states and union territories.
- Government schools contribute significantly to the total number of schools.
- Enrollment patterns differ across school management types.

## Project Objective
The goal of this project is to analyze educational indicators and present them through interactive visualizations to make data easier to understand and explore.
